Design System Notes

  • Fonts: mOS Black (UI/buttons/labels) + Angelinas (handwriting/scripty text)
  • Golden Ratio: All spacing follows PHI (1.618) multiplier system
  • Start Button: GREEN #51CF66 across all levels
  • Error Color: RED #E22237 (from Excalidraw palette)
  • Text Contrast (UX-First):
    • Level 1: DARK (#2C3E50) on light yellow canvas - high contrast
    • Level 2: GOLD (#FFD700) on dark blue body - high contrast
    • Level 3: WHITE (#FFFFFF) on green nature canvas - high contrast
  • Level Atmospheres PRESERVED:
    • Level 1: Yellow/gold (Carnegie's bright greeting hall)
    • Level 2: Blue/gray (Bach's sophisticated arrangement room)
    • Level 3: GREEN nature (Feynman's living gallery)
  • UI Components: Rough.js hand-drawn style (rough-ui.js library)
  • Purple Accent: #5E35B1 for handwritten notes (from Level 2)
  • Layout Parts:
    • HEADER: Level title (Angelinas) + Subtitle (mOS Black)
    • UI LAYER: Rough.js buttons, panels, HUD (hand-drawn)
    • GAME CANVAS: Score displays, progress bars, game objects
    • FOOTER: Handwritten notes/hints (Angelinas)
  • Level 4 Added: Purple atmosphere (Turing's Question Corner)
